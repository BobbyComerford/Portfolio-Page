Activity-Space Simulation is a data-driven approach to modelling working spaces for optimal productivity. The service, provided by Cambridge Architectural Research, required a website which educated and engaged visitors.

The content display flexes through user manipulation and the content is infused with interactive elements, mimicking the service’s ideology to deliver unique user experiences. Rollover gifs play with type to visually emphasise key messages, while an algorithm-led optimal capacity calculator provides a taster of Activity-Space Simulation.
